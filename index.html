Popsicle JS

Download Popsicle JS

Popsicle JS is a small-footprint touch event library I built as an alternative to jQuery Mobile and Hammer JS. Popsicle is written in raw JavaScript so it doesn't require jQuery (although it does work with jQuery) and there are no polyfills for features implemented by all modern browsers.

Popsicle provides the following custom touch events:

  • onefingertap
  • swipeup
  • swipedown
  • swipeleft
  • swiperight
  • twofingertap
  • threefingertap
  • fourfingertap
